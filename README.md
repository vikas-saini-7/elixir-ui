## ğŸš€ Quick Start

### Installation

```bash
# Initialize your project with Elixir UI
npx elixir-labs-ui init

# Add your first component
npx elixir-labs-ui add button
```

### Usage

```jsx
import { Button } from "./components/ui/button";

export function App() {
  return (
    <Button variant="default" size="md">
      Click me
    </Button>
  );
}
```
